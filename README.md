# 📊 Análisis de Facturación

Este proyecto tiene como objetivo analizar el comportamiento de la facturación a partir de un dataset real, utilizando herramientas de análisis de datos y visualización.

## 🧹 Limpieza y preparación de datos
Se realizaron tareas de limpieza sobre el dataset, incluyendo:
- Conversión de variables a formatos adecuados (fecha, numérico)
- Eliminación de valores nulos en variables críticas
- Ajuste de variables categóricas para facilitar el análisis

## 📈 Exploración y visualización de datos
Se analizaron distintos aspectos de la facturación:
- Evolución temporal de los importes
- Distribución por estado de los comprobantes
- Identificación de principales clientes
- Análisis por reparto

## 📊 Dashboard en Power BI
Se desarrolló un dashboard interactivo que permite visualizar los resultados de forma clara, incluyendo:
- Total facturado
- Cantidad de comprobantes emitidos
- Cantidad de clientes facturados
- Filtros por período

## 🧠 Conclusión
El análisis permitió identificar patrones en la facturación, destacando la concentración en ciertos clientes y repartos, así como un alto porcentaje de comprobantes en estado pagado.
